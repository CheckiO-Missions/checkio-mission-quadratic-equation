Квадратне рівняння може бути представлене як добуток

(x-x1)(x-x2) = 0,
де x1 and x2 це розв'язки рівняння, так звані корені. Після розкриття дужок, ти отримаєш відому форму
a*x**2 + b*x + c = 0.
Це те, що тобі потрібно зробити в цій місії. Ти маєш вхідний {% if is_js %}масив{% else %}список{% endif %} з цілими числами - a, x1 [, x2]. Якщо він має два елементи, це значить x1 == x2: рівняння має два однакові корені (один унікальний корінь). Твоя функція повинна повертати квадратне рівняння як рядок. Приділи увагу специфічним випадкам. Використай калькулятор для перевірки. Удачі!

Вхідні значення: {% if is_js %}Масив{% else %}Список{% endif %} з 2-3 цілими числами.

Вихідні значення: Рядок.

Як це використовуються: Практика представлення рівнянь в комфортному для сприйняття вигляді.

Передумови:

  • a != 0; abs(a) == 1 -> '[-]x**2'; a != abs(1) - > '[-]a*x**2'
  • строго один пробіл між знаками: [-]a*x**2 знак b*x знак c = 0
  • abs(b) == 1 -> [-]a*x**2 знак x знак c = 0
  • слідкуй за правильним виглядом і пробілами при x1=x2=0, x1 or x2 = 0, x1=-x2
